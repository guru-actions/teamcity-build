== Usage example

In your YAML file, add:

[source,yaml]
----
      - name: Install Helm chart
        uses: cloudbees-io/helm-install@v1
        with:
          release-name: nginx
          chart-location: oci://ghcr.io/nginxinc/charts/nginx-ingress

----
